# 德州扑克游戏 (Texas Hold'em Poker)

在 VS Code 中玩德州扑克游戏的扩展！

## 功能特性

- 🎴 完整的德州扑克游戏逻辑
- 🤖 智能机器人对手 (最多 6 个)
- 💰 实时筹码和底池管理
- 🎯 状态栏游戏控制
- ⚙️ 可自定义游戏参数
- 📊 游戏状态和历史记录查看

## 安装方法

1. 在 VS Code 扩展市场搜索 "Texas Hold'em Poker"
2. 点击安装
3. 重启 VS Code

## 使用方法

### 开始游戏

1. 在状态栏点击 "开始游戏" 按钮
2. 或按 `Ctrl+Shift+P` 输入 "开始游戏"

### 游戏操作

- **开始游戏**: 开始新的一局
- **弃牌**: 放弃当前手牌
- **跟注**: 跟上当前赌注
- **加注**: 输入加注金额
- **快速加注 50**: 快速加注 50 筹码

### 设置游戏参数

- 在状态栏悬停，点击日志窗口中的 "设置游戏参数"
- 可以设置：
  - 小盲注金额
  - 大盲注金额
  - 玩家初始筹码
  - 玩家数量 (2-9 人)

### 游戏规则

- 遵循标准德州扑克规则
- 支持翻牌、转牌、河牌三个阶段
- 智能 AI 对手决策
- 自动计算胜负和筹码分配

## 状态栏显示

- **游戏进行中**: 显示当前筹码、底池、手牌、公牌等
- **等待操作**: 显示可用的操作按钮
- **游戏结束**: 显示胜负结果和重新开始选项

## 要求

- VS Code 1.74.0 或更高版本

## 扩展设置

目前所有设置都通过状态栏的设置按钮进行配置。
